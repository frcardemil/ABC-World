# ABC-World

¡Bienvenido a ABC-World!

El presente proyecto consiste en un juego educativo digital orientado a niños y niñas de pre kínder y kínder, con el propósito de fomentar el aprendizaje inicial de la lectura y el reconocimiento de letras a través de una experiencia lúdica, didáctica e innovadora.

## Visualización

Para poder visualizar e interactuar con el juego, por favor ingrese al siguiente link: https://abcworldmovil.web.app

## Tecnología Usada

Este proyecto está construido utilizando las siguientes tecnologías:

* **Backend:** Node.js, Express
* **Base de Datos / Hosting:** Firebase
* **Frontend:** HTML, CSS y JavaScript
